# [SiYuan](https://github.com/siyuan-note/siyuan)

### Info
> "SiYuan is a privacy-first personal knowledge management system, support fine-grained block-level reference and Markdown WYSIWYG."

### Options

#### 1. Firstly check the [common web services options](../web_options.md)
#### 2. Specific options for this module:

`siyuan.admin-password`:
Default password for the admin

`siyuan.paths`:
| Name    | Description              | Default                   |
| ------- | ------------------------ | ------------------------- |
| default | The main path of the app | `<main path>/siyuan` |

- `<main path>` = Main path for all the apps. See [defaults](../defaults.md#paths).
